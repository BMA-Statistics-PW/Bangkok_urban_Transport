# Bangkok Urban Transport Dashboard

**Dashboard สถิติผู้โดยสารระบบขนส่งสาธารณะ กรุงเทพมหานคร ปี 2556–2568**

จัดทำโดย กลุ่มงานสถิติและวิจัย กองนโยบายและแผนงาน สำนักการจราจรและขนส่ง กรุงเทพมหานคร เพื่อใช้เป็นเครื่องมือสนับสนุนการวิเคราะห์ วางแผน และกำหนดนโยบายด้านการขนส่งสาธารณะในเขตกรุงเทพมหานครและปริมณฑล

เว็บไซต์ : <https://bma-statistics-pw.github.io/Bangkok_urban_Transport/>

---

## วัตถุประสงค์

เสนอข้อมูลสถิติผู้โดยสารระบบขนส่งสาธารณะของกรุงเทพมหานครในรูปแบบที่เข้าใจง่าย เปรียบเทียบได้ และสนับสนุนการตัดสินใจเชิงนโยบาย โดยรวบรวมข้อมูลจากหน่วยงานที่เกี่ยวข้องทั้งหมดไว้ในที่เดียว

## ขอบเขตข้อมูล

- **ช่วงเวลา :** ปี พ.ศ. 2556–2568 (ค.ศ. 2013–2025)
- **หน่วย :** ล้านเที่ยวคน/ปี (ข้อมูลผู้โดยสาร) และ ล้านคน-เที่ยว/ปี (ข้อมูลสัดส่วนการเดินทาง)
- **พื้นที่ศึกษา :** กรุงเทพมหานครและปริมณฑล

## ระบบขนส่งที่ครอบคลุม

| ประเภท | ระบบ |
|---|---|
| รถโดยสารประจำทาง | ขสมก., BRT |
| รถไฟฟ้า | BTS สายสีเขียว, BTS สายสีทอง, MRT สายสีน้ำเงิน/ม่วง/เหลือง/ชมพู, ARL, SRT สายสีแดง |
| เรือโดยสาร | เรือข้ามฟาก, เรือด่วนเจ้าพระยา, คลองแสนแสบ, เรือไฟฟ้า, เรือภัตตาคาร |

## โครงสร้างแดชบอร์ด

แดชบอร์ดแบ่งการนำเสนอออกเป็น 5 ส่วน ได้แก่

1. **ภาพรวม** — ตัวชี้วัดสำคัญ (KPI) ปีล่าสุด แนวโน้มผู้โดยสารทุกระบบ และสัดส่วนผู้โดยสารรายระบบ
2. **ผู้โดยสารรายสาย** — กราฟแนวโน้มรายปีของรถไฟฟ้า รถโดยสาร และเรือ พร้อมตารางข้อมูลรายสาย
3. **สัดส่วนรูปแบบ** — สัดส่วนการเดินทางระหว่างระบบขนส่งสาธารณะกับรถส่วนบุคคล (อ้างอิงแบบจำลอง BTDM ของ สนข.)
4. **รายปี** — ภาพรายละเอียดของแต่ละปีแบบเลือกดูได้
5. **แหล่งข้อมูล** — รายการไฟล์ แหล่งที่มา และข้อจำกัดของข้อมูล

## แหล่งข้อมูล

- สำนักการจราจรและขนส่ง กรุงเทพมหานคร (สจส.)
- สำนักงานนโยบายและแผนการขนส่งและจราจร (สนข.) — แบบจำลอง BTDM ปีฐาน 2565
- องค์การขนส่งมวลชนกรุงเทพ (ขสมก.)
- การรถไฟฟ้าขนส่งมวลชนแห่งประเทศไทย (รฟม.)
- การรถไฟแห่งประเทศไทย (รฟท.) และ บริษัท รถไฟฟ้า ร.ฟ.ท. จำกัด (รฟฟท.)
- กรมการขนส่งทางราง และ กรมการขนส่งทางบก
- กรมเจ้าท่า

## ข้อจำกัดและข้อควรระวัง

1. หน่วยข้อมูลแตกต่างกัน : ไฟล์ผู้โดยสารใช้ "เที่ยวคน" ส่วนไฟล์สัดส่วนใช้ "คน-เที่ยว/ปี" (ปรับด้วย trip rate)
2. ขสมก. ใช้ **ปีงบประมาณ** (ต.ค.–ก.ย.) ขณะที่ระบบรางใช้ **ปีปฏิทิน** (ม.ค.–ธ.ค.) การเปรียบเทียบต้องพิจารณาความแตกต่างนี้
3. ปี 2563–2564 ได้รับผลกระทบจากสถานการณ์ COVID-19 ตัวเลขต่ำกว่าปกติ ไม่ควรนำไปเปรียบเทียบกับปีอื่นโดยตรง
4. BRT ปี 2567 ไม่มีข้อมูลเนื่องจากอยู่ระหว่างปรับเปลี่ยนระบบการให้บริการ
5. สัดส่วนการเดินทางปี 2567 (11.7%) ต่ำกว่าปี 2566 เนื่องจากปรับวิธีคำนวณ (methodology) มิใช่ผู้โดยสารลดลงจริง
6. ข้อมูลไม่ครอบคลุม : รถตู้โดยสาร รถไฟระหว่างเมือง รถสองแถว และมอเตอร์ไซค์รับจ้าง
